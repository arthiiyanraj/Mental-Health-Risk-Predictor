# Mental Health Risk Predictor

A machine learning web application that predicts mental health risk levels based on user inputs.

# Features
- Predicts **Low / Medium / High** risk levels
- Inputs: Age, Sleep Hours, Screen Time, and more
- Built with Flask + Scikit-learn

# Dataset
- Dataset sourced from Kaggle

# Tech Stack
- Python, Flask, Scikit-learn, Pandas
- HTML/CSS (Frontend)

# How to Run
```bash
pip install flask scikit-learn pandas
python app.py
```
Then open http://localhost:5000
